git 형상관리를 이용한 프로젝트 관리 세미나 및 팀빌딩 발대식

2025년 06월 12일 라마다플라자 광주호텔, 5F 대연회장(광주 서구 상무자유로 149)에서 팀빌딩 역량강화 및 네트워크, 비용처리 안내를 위한 세미나와 팀빌딩 발대식이 개최되었습니다.

현 시점 셜LOCK에 활동 중인 모든 팀원이 참가하였으며, GIT을 이용한 형상관리*와 비용처리 방법을 교육받았습니다.

오늘 세미나 내용을 바탕으로 형상관리 및 비용처리에 대한 이해도를 높이고, 실제 프로젝트에 적극적으로 적용할 계획입니다.

특히 GIT을 활용한 형상관리 교육은 팀 내 협업 시 버전 충돌을 방지하고, 작업 이력을 체계적으로 관리하는 데 큰 도움이 될 것으로 기대됩니다. 각 팀원이 실습을 통해 직접 commit, branch, merge 등의 과정을 경험함으로써, 앞으로의 개발 업무에서 효율성과 일관성을 높일 수 있을 것입니다.

또한, 비용처리 절차에 대한 안내를 통해 프로젝트 진행 중 발생하는 비용을 정확하게 기록하고 관리하는 방법을 숙지하였습니다. 실제 운영 단계에서 예산 집행 전 우선적으로 팀빌딩 담당자와 검토하는 작업을 선행하도록 합시다.

이번 팀빌딩 발대식을 계기로 팀원 간의 유대감도 한층 강화되었으며, 향후 협업 과정에서도 오늘 배운 내용을 기반으로 보다 책임감 있게 임할 수 있을 것으로 기대됩니다.


*형상관리 : 소프트웨어의 변경사항을 체계적으로 추적하고 통제하는 것
  • Git 사용 방법에 대해 점검 필요 (gitLab 위주 설명이지만 Local 명령어 중심으로 확인 가능)
  • GitHub 미사용자를 대상으로 1차 교육 실시 예정
  • 역량 강화 세미나 관련 PPT 메일공유가 시행됐는지 수시로 확인
  • 지출 증빙용 영수증 반드시 첨부
  • 동일날짜 AI랩실 회의록 작성하기
